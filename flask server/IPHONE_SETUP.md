# iPhone 離線留言安裝

## 第一次安裝

1. 在 Mac 執行 `./start.sh`。
2. 確認 iPhone 與 Mac 連接同一個 Wi-Fi。
3. 用 iPhone Safari 開啟啟動訊息中的「憑證網址」（連接埠 `5003`），下載 CA 憑證。
4. 到「設定」→「一般」→「VPN 與裝置管理」，安裝 `Teacher Messenger Local CA`。
5. 到「設定」→「一般」→「關於本機」→「憑證信任設定」，開啟該憑證的完整信任。
6. 回到 Safari，開啟啟動訊息中的 HTTPS 手機網址。
7. 點選「分享」→「加入主畫面」。
8. 至少開啟主畫面 App 一次，等待頁面顯示「已連線至老師的電腦」。

## 使用方式

- Mac 可連線時，留言會立即傳送。
- Mac 關機、休眠或網路中斷時，留言會保存在 iPhone。
- iPhone 再次連到 Mac 後，頁面會自動補送待傳留言。
- 瀏覽器不能執行真正的 ICMP ping；本專案以 `/health` HTTP 請求判斷伺服器是否可用。

## 注意事項

- 請從主畫面的 App 開啟，不要從「檔案」App 開啟 HTML。
- 若 Mac 的區域網路 IP 改變，重新執行 `./start.sh` 會重建伺服器憑證；CA 憑證不需重裝。
- 請勿刪除 Safari 網站資料，否則尚未傳送的手機留言可能被清除。

\subsection*{混合題參考}
0 至 3 秒位移為 $+4\times 3=+12\ \mathrm{m}$；3 至 7 秒位移為 $-2\times 4=-8\ \mathrm{m}$，總位移為 $+4\ \mathrm{m}$。路徑長為 $12+8=20\ \mathrm{m}$。平均速度為 $+4/7\ \mathrm{m}/\mathrm{s}$。平均速率為 $20/7\ \mathrm{m}/\mathrm{s}$；兩者不同是因為平均速度使用有方向的位移，平均速率使用實際走過的路徑長。

# 高一段考出題風格與記憶點

## 整體風格

- 單選題常用「何者錯誤」或「何者正確」檢查基本概念。
- 複選題常指定應選幾項；第三次段考開始出現不提示應選幾項的分科測驗風格。
- 題目偏好生活化情境：遊樂設施、半導體、指北針、AI 能源、太陽能、極光、皮膚曬傷、福島核災。
- 常把章節觀念串接在同一題，例如電磁感應與能量守恆、波動與量子、原子結構與基本交互作用。
- 圖像題不一定重計算，常考判讀原則：斜率、面積、正負號、折返、磁通量變化。

## 高頻迷思

- SI 單位與導出量拆解錯誤。
- 現行 SI 改用物理常數定義後，誤以為未來不可能修訂。
- 馬克士威、赫茲、厄斯特、法拉第的角色混淆。
- 位置－時間圖折返與速度－時間圖折返判斷混淆。
- 平均速度量值與平均速率混淆。
- 合力為零與位移必定正比於時間的敘述過度推論。
- 靜摩擦力被誤認為固定大小。
- 向心力被誤認為一種新的力。
- 克卜勒第三定律忘記使用半長軸。
- 有磁場就一定有感應電流。正確關鍵是磁通量改變。
- 電磁波需介質、或馬克士威已實驗發現電磁波。
- 強核力、弱作用、電磁力混淆。
- 有摩擦或非保守力時，誤以為總能量守恆不成立。
- 光強度提高即可產生光電效應，或強度提高會增加最大動能。
- 物質波被誤認為介質波。
- 波耳模型被誤認為完整現代原子模型。

## 常考計算模型

- 等加速度：`v = v0 + at`、位移面積、自由落體時間。
- 大怒神／電梯／拋落物：先判斷是否為慣性系，再用相對運動或自由落體。
- 萬有引力比較：`F = GmM/r^2`，距離平方影響大。
- 克卜勒第三定律：`T^2 proportional a^3`。
- 轉移橢圓：半長軸取內外軌道半徑平均，再用第三定律估週期。
- 功：`W = Fd cos theta`；垂直力不作功。
- 動能與光子能量：`K = (1/2)mv^2`、`E = hf = hc/lambda`。
- 核能：質量虧損 `E = mc^2`，再接熱量 `Q = mc Delta T`。
- 能階：最大光子能量對應最大能階差，最短波長對應最大頻率。

## 出題設計可複用方向

- 用真實新聞或科技情境包裝基本概念，但選項回到單一物理判斷。
- 將物理史人物設計成配對題，特別是厄斯特、法拉第、馬克士威、赫茲、普朗克、愛因斯坦、德布羅意、波耳。
- 用「錯誤迷思選項」檢查學生是否過度背誦，例如「有磁場就有感應電流」、「電子繞核靠強核力」。
- 選修銜接題可從高一版本問起，再加入進階量如截止電壓、物質波波長、角動量量子化。
- 對小考而言，可將段考題改寫成純文字題，不必保留圖片；圖形可用文字描述讓老師手畫。

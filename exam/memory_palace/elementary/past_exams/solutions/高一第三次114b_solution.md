## 重點解題

- 第 3 題：功的單位是焦耳，`W = Fd cos theta`。正向力或重力是否作功，要看力與位移方向。
- 第 4-6 題：功、能量與功率題常用「同一距離」、「不同質量」、「不同高度」比較，先確認能量變化量，再判斷功率或時間。
- 第 7-8 題：核能與放射性題常混合半衰期、劑量、alpha/beta/gamma 衰變與核種變化。alpha 衰變會使質量數少 4、原子序少 2。
- 第 12-13 題：太陽能光電轉換與光電效應相關，光子能量需高於材料能隙或功函數才有效。
- 第 17-18 題：極光與太陽暗線皆與原子能階有關。發射光譜是高能階降到低能階放光；吸收暗線是低能階吸收特定光子躍遷到高能階。
- 第 19 題：發射光譜波長最短代表光子能量最大，對應能階差最大。
- 第 21 題：多普勒效應以相對接近或遠離判斷觀測頻率大小。
- 第 22 題：作功為零可由力與位移垂直或保守力封閉路徑總功判斷。
- 第 26 題：若綠光可產生光電子，頻率更高的藍、紫光也可；紅光頻率較低，不一定可產生。光電子最大動能由頻率決定。
- 第 27 題：`E = hf`，`f = 5.5 x 10^-19 / 6.6 x 10^-34 ≈ 8.3 x 10^14 Hz`，`lambda = c/f ≈ 3.6 x 10^-7 m = 360 nm`，比可見光短，屬紫外線。
- 第 28-29 題：電子雙狹縫顯示電子具有波動性；物質波不是介質波，動量越大波長越短。
- 第 30 題：波耳模型中電子可在特定穩定軌道不輻射；能階躍遷造成吸收或發射光譜，但模型不是完整現代量子力學。

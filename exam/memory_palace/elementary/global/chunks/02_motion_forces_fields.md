# 運動、力與場

## 描述運動

- 位置描述物體在哪裡；位移描述位置改變，屬向量。
- 速度描述位置改變率；加速度描述速度改變率。
- 等加速度直線運動常用公式包含 `v = v0 + at`、`x = x0 + v0 t + (1/2)at^2`、`v^2 = v0^2 + 2a Delta x`。
- 圖像判讀：位置－時間圖斜率是速度；速度－時間圖斜率是加速度，面積是位移。

## 從亞里斯多德到伽利略

- 亞里斯多德把運動分為自然運動與強制運動，並傾向認為維持運動需要作用。
- 伽利略強調實驗與推理，指出若沒有外力或阻力，物體可保持等速直線運動；落體快慢不由重量單獨決定。
- 這條思想線通往牛頓第一運動定律。

## 牛頓三定律

- 第一定律：物體若不受合外力，保持靜止或等速直線運動。
- 第二定律：合力造成加速度，`F_net = ma`。
- 第三定律：作用力與反作用力大小相等、方向相反、作用在不同物體上。

重要判斷：

- 作用力與反作用力不會互相抵消，因為它們作用在不同物體上。
- 判斷物體運動要看該物體所受合力，不是看單一作用力對。
- 馬拉車題：馬拉車與車拉馬是一對作用反作用；馬車系統能前進，是因為地面對馬的摩擦力提供系統外力。

## 常見力

- 重力：近地表常用 `mg`。
- 正向力：接觸面垂直作用於物體，不一定等於重量。
- 彈力：彈簧常用虎克定律 `F = kx`。
- 摩擦力：方向反抗相對運動或相對運動趨勢；最大靜摩擦力與正向力相關，動摩擦力常近似 `mu_k N`。

## 天體運動

- 克卜勒第一定律：行星繞太陽軌道為橢圓，太陽位於一焦點。
- 克卜勒第二定律：行星與太陽連線在相等時間掃過相等面積。
- 克卜勒第三定律：繞同一中心天體時，週期平方與半長軸三次方成正比。
- 牛頓萬有引力可解釋克卜勒定律，連回 Ch2 的重力交互作用。

- Ch2：重力與電磁力是許多力的根源。

# 跨章節橋接與出題切入點

## 原子結構到量子

題目可從拉塞福金箔散射開始，追問：

- 為什麼可推論原子核很小、帶正電、質量集中？
- 為什麼拉塞福模型用古典電磁學會不穩定？
- 波耳如何用定態與能階解釋原子光譜？

核心連線：Ch2 原子模型 -> Ch6 波耳模型 -> Ch6 原子光譜 -> Ch5 能量守恆。

## 力學到能量

同一情境可要求兩種分析：

- 用牛頓第二定律求加速度或受力。
- 用功能定理或能量守恆求速度、高度或能量轉換。

核心連線：Ch3 力與運動 -> Ch5 功與能量。

## 重力到天體

克卜勒定律可作為觀察規律，牛頓萬有引力可作為解釋機制。

核心連線：Ch2 重力交互作用 -> Ch3 克卜勒與牛頓 -> Ch5 重力位能。

## 電磁到光

電流磁效應、電磁感應、馬克士威方程與電磁波是同一條發展線。

核心連線：Ch4 厄斯特 -> 法拉第 -> 馬克士威 -> 光是電磁波 -> Ch6 光電效應。

## 波動到量子

同一個「光」在不同實驗中展現不同性質：

- 雙狹縫干涉、繞射：波動性。
- 光電效應：粒子性。
- 原子光譜：光子能量與能階差。

核心連線：Ch4 波動 -> Ch6 光子 -> Ch5 能量量子。

## 核能到基本作用

核分裂與核融合不能只記反應名，要連到：

- 原子核尺度與核子。
- 強核力與電磁斥力的競爭。
- 質量虧損與 `E = mc^2`。
- 能量守恆與能量轉換效率。

核心連線：Ch2 強作用與原子核 -> Ch5 核能 -> Ch1 數量級。

# 量子現象、光子與原子模型

## 量子化

- 量子化指某些物理量只能取離散值，有最小不可再分的單位或特定允許值。
- 電荷量是基本電荷的整數倍。
- 普朗克提出能量量子化，開啟近代物理。

## 光電效應

現象：

- 光照射金屬表面，使電子逸出，逸出的電子稱光電子。
- 若形成電流，稱光電流。

實驗重點：

- 是否能產生光電效應，關鍵是入射光頻率是否高於該金屬底限頻率。
- 頻率低於底限頻率時，強度再大也不能產生光電子。
- 頻率足夠時，幾乎立即產生光電子，沒有古典理論預期的等待時間。
- 頻率固定且高於底限頻率時，增加光強度會增加光電子數或光電流，但不增加光電子最大動能。
- 提高頻率會增加光電子最大動能。

光子模型：

- 光子能量 `E = hf = hc/lambda`。
- 頻率越高，單一光子能量越大；波長越短，單一光子能量越大。
- 光強度可理解為單位時間抵達的光子數較多。
- 每個電子一次吸收一個光子或不吸收；若光子能量不足，不會靠長時間累積多個低能光子而逸出。

## 波粒二象性

- 光的干涉與繞射顯示波動性。
- 光電效應顯示粒子性。
- 電子繞射、電子雙狹縫實驗顯示物質也有波動性。
- 同一實驗安排通常不會同時觀察到完整的波動性與粒子性，這是互補觀點。

## 原子光譜與波耳模型

- 原子光譜是不連續譜線，可作為元素指紋。
- 同一元素的發射光譜與吸收光譜譜線位置相對應。
- 拉塞福模型無法解釋原子穩定與不連續光譜。
- 波耳模型假設電子只能在特定定態軌道，不在定態中連續輻射能量。
- 電子在能階間躍遷時，吸收或放出光子，滿足 `hf = Delta E`。
- 光子吸收必須剛好符合能階差；碰撞激發時，入射粒子動能只要足夠，多餘能量可保留為動能。

## 跨章連結

- Ch2：拉塞福原子核模型與原子組成是波耳模型的前置知識。
- Ch4：光先作為電磁波處理，再由光電效應補上粒子性。
- Ch5：所有光電效應與能階躍遷題都要回到能量守恆。

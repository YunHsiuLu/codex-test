# 電、磁、電磁波與波動

## 電流磁效應

- 厄斯特發現電流會產生磁效應，建立電與磁的連結。
- 直導線、圓形線圈、螺線管的磁場方向可用右手定則判斷。
- 圓形線圈中心磁場方向：右手四指沿電流方向，拇指指向線圈中心軸上的磁場方向。

## 電磁感應

- 法拉第發現：封閉線圈中的磁通量隨時間改變時，會產生感應電流或感應電動勢。
- 產生感應電流的關鍵是「磁通量改變」，不是單純有磁場。
- 冷次定律：感應電流方向會反抗造成磁通量改變的原因。
- 電磁感應遵守能量守恆；發電機輸出的電能來自外界做功或其他能量轉換。

## 馬克士威與電磁波

- 馬克士威統一電與磁，指出變動電場可產生磁場，變動磁場可產生電場。
- 電磁波不需要介質，可在真空中傳播。
- 光是電磁波的一種。電磁波譜包含無線電波、微波、紅外線、可見光、紫外線、X 光、伽瑪射線等。

## 波動基本量

- 週期 `T`：完成一次振動所需時間。
- 頻率 `f`：每秒振動次數，`f = 1/T`。
- 波長 `lambda`：相鄰同相位點的距離。
- 波速 `v = f lambda`。
- 振幅與波傳遞能量相關。

## 波動現象

- 波傳遞能量與擾動，不傳遞介質本身的長距離位移。
- 折射：波進入不同介質時速度改變而方向改變。
- 繞射：波遇障礙或狹縫後彎入陰影區，狹縫越接近波長越明顯。
- 干涉：兩波重疊造成增強或減弱。
- 多普勒效應：波源與觀察者相對運動時，觀察頻率改變。

# 出題設計矩陣

## 基礎題素材

- 單位與數量級：原子 `10^-10 m`、原子核 `10^-15 m`、`1 kWh = 3.6 x 10^6 J`。
- 人物配對：拉塞福、湯姆森、波耳、伽利略、牛頓、克卜勒、厄斯特、法拉第、馬克士威、普朗克、愛因斯坦。
- 公式判斷：`F = ma`、`W = Fd cos theta`、`K = (1/2)mv^2`、`v = f lambda`、`E = hf`。
- 圖像判讀：位置－時間圖、速度－時間圖、波形圖、能階圖、光電效應圖。

## 進階題素材

- 比較題：光強度與光頻率對光電效應的不同影響。
- 多概念題：電磁感應方向與能量守恆。
- 跨尺度題：原子尺度、原子核尺度、可見光波長、光學顯微鏡限制。
- 現象解釋：摩擦力作功使機械能降低但總能量守恆。
- 模型限制：拉塞福模型為何需要波耳模型補充。

## 常見迷思選項

- 強度夠大時，任何頻率的光都能產生光電效應。
- 有摩擦力時，能量守恆律不成立。
- 原子質量主要來自電子。
- 原子核大小與原子大小同量級。
- 作用力與反作用力會因大小相等而抵消。
- 電磁感應只要有磁場就會產生感應電流。
- 電磁波一定需要介質。
- 波會把介質從波源一路帶到遠方。
- 四種基本交互作用都遵守平方反比。

## 混合題方向

- 給一段科學史短文，要求判斷人物、發現、模型限制與後續理論。
- 給一張能階圖，問可吸收或放出的光子頻率，並連到光譜。
- 給發電機或線圈磁鐵情境，問感應電流方向、能量來源與冷次定律。
- 給滑面或斜面情境，要求同時判斷受力、加速度、功與能量轉換。
- 給光電效應實驗，改變頻率或強度，問光電流與最大動能如何變化。

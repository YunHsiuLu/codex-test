# Chunk 01：科學態度與科學方法

來源頁面：page-000001、page-000002

## 核心概念

科學態度：

- 理性思考。
- 不固執己見。
- 不盲從權威或多數意見。
- 不妄下決斷。
- 科學理論沒有絕對權威，必須經由觀察或實驗檢驗後才可成立並被接受。
- 當新證據與舊理論衝突時，理論可能需要修正。

科學方法流程：

1. 提出問題。
2. 形成假設。
3. 設計實驗。
4. 收集資料及分析數據。
5. 得出結論。
6. 表達分享。

物理學研究原則：

- 將自然界複雜現象背後隱藏的結構秩序，用簡單清晰的數學規律描述。
- 以客觀實驗檢視推論結果。

## 重要例子

愛因斯坦廣義相對論：

- 1915 年發表廣義相對論。
- 預言光的前進方向會因重力而彎曲。
- 1919 年愛丁頓爵士日全食測量結果支持此推論，之後廣義相對論才被接受。

這個例子常用來考：

- 科學理論需要可檢驗預測。
- 觀測與實驗證據是科學接受理論的重要依據。
- 理論不是只靠權威或數學漂亮就成立。

## 常見誤區

- 錯：課本寫了或權威說了就是真理。
- 錯：多數人相信即可視為科學定律。
- 錯：不合假設的資料可以刪除。
- 錯：科學理論一旦提出就不會修正。

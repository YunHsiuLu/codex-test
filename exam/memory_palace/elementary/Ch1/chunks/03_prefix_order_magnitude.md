## 科學記號

測量數值 Z 可寫成：

`Z = a x 10^n`，其中 `1 <= a < 10`，n 為整數。

## 數量級

粗略數量級：

- 指數 n 相差 1，表示大約大 10 倍或小 10 倍。
- `10^n` 可作為物理量的數量級。

更精確的數量級判斷：

- 若 `a < sqrt(10) ~= 3.16`，則 `Z` 的數量級為 `10^n`。
- 若 `a >= sqrt(10) ~= 3.16`，則 `Z` 的數量級為 `10^(n+1)`。

例：

- `1.6 x 10^-19` 的數量級為 `10^-19`。
- `6.67 x 10^-11` 的數量級為 `10^-10`。

## 常見換算

- `1 km = 10^3 m`
- `1 ms = 10^-3 s`
- `1 μs = 10^-6 s`
- `1 μA = 10^-6 A`
- `1 GHz = 10^9 Hz`
- `1 MHz = 10^6 Hz`
- `1 cm = 10^-2 m`
- `1 nm = 10^-9 m`
- `1 Å = 10^-10 m`
- `1 TB = 10^12 B`
- `1 GB = 10^9 B`

範例：

- `103.3 MHz = 1.033 x 10^8 Hz`
- `1 ly = 9.46 x 10^15 m = 9.46 x 10^12 km`
- `0.53 Å = 5.3 x 10^-5 μm`
- `250 MHz = 2.5 x 10^8 Hz`
- `10 nm = 100 Å`
- `1 TB / 2 GB = 500`

## googol 與大數

- `1 googol = 10^100`，即 1 後面有 100 個 0。
- 宇宙原子數量級約 `10^78` 到 `10^84`，小於 googol。
- 19 x 19 圍棋棋盤 361 點，各點可有白子、黑子、不放子三狀態，總狀態 `3^361`，以 10 為底約 `10^172`，大於 googol。

## 奈米尺度與表面效應

奈米：

- nano 源自希臘文，本意類似矮子或侏儒。
- `1 nm = 10^-9 m`。
- 奈米技術研究尺度約 1 到 100 nm。
- 奈米粒子通常指尺寸 1 到 100 nm 的粒子。

表面效應：

- 顆粒縮小時，表面積／體積比急劇增加。
- 直徑大於 100 nm 的顆粒，表面效應常可忽略。
- 小於 100 nm 時，表面原子比例上升，材料會呈現特殊物理性質。
- 球體表面積 `A = 4πr^2`，體積 `V = 4/3πr^3`，所以 `A/V` 與 `1/r` 成正比。
- 半徑從 100 nm 縮小到 5 nm，`A/V` 放大 20 倍。

素養題素材：

- 將海灘球與地球尺度類比，可用比例換算奈米粒子尺寸。
- PM2.5 是粒徑小於或等於 2.5 μm 的懸浮微粒。

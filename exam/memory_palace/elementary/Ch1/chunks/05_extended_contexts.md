## SI 基本單位修訂的精神

SI 基本單位的定義會隨科技進步持續修訂。修訂思路：

- 找出宇宙中不變的物理常數。
- 盡可能將常數測量精準。
- 將 SI 單位定義與這些物理常數建立關聯。

長度例子：

- 1983 年，1 公尺定義為光在真空中經過 `1/299792458` 秒所行的距離。
- 因真空中光速 `c = 299792458 m/s` 是不變量。
- 提高光速測量精度，可提升長度單位精度。

2018 年第 26 屆國際度量衡大會調整：

- 公斤、安培、克耳文、莫耳的定義與基本物理常數建立關聯。

- 2018 年 SI 基本單位「公斤」與普朗克常數建立關聯。
- 克耳文與波茲曼常數建立關聯。

## 克耳文與水三相點

舊定義：

- 克耳文以熱力學為基礎。
- 1 K 原本定義為水三相點熱力學溫度的 `1/273.16`。
- 水三相點：水蒸氣、水、冰三種物態同時存在的唯一溫度。
- 該溫度為 273.16 K，約 0.01°C。

精密測量偏差來源：

- 不同國家三相點管內的水可能由不同比例氫、氧同位素組成。
- 長時間後玻璃會溶入水中形成雜質。
- 玻璃管本身成分與雜質也會影響三相點溫度。

波茲曼常數與溫度：

- 波茲曼常數 k 連繫絕對溫度與能量。
- 理想氣體分子平均移動動能與絕對溫度成正比：
  `E_k = 3/2 kT`
- 若 `E_k = 6.21 x 10^-21 J`，取 `k = 1.38 x 10^-23 J/K`，則 `T = 300 K`。

## 歷屆試題素材

常見題型：

- 科學史正誤：愛因斯坦以光子模型解釋光電效應；馬克士威不能解釋光電效應。
- 導出單位判斷：功率 `kg m^2/s^3` 正確。
- 能量單位：`kg m^2/s^2`。
- 科學家與成果配對：克卜勒對應行星運動定律；都卜勒對應相對運動造成頻率改變；馬克士威對應電磁波。
- 外星單位密度換算：密度 `D=M/V`，若質量與長度單位換算已知，代入比例即可。

## 可延伸為素養題的脈絡

- SI 單位修訂：從具體物件或物態標準，轉向宇宙常數。
- 奈米表面效應：尺度縮小導致表面積／體積比上升。
- PM2.5：粒徑尺度換算與健康影響。
- Googol 與圍棋盤：數量級比較與科學記號。

# Ch1 Question Bank Seeds

此檔不是正式題庫，而是未來命題時可快速抓取的題型種子。

## 基礎題種子

1. 科學態度判斷：選出符合「理性、實驗檢驗、不盲從」的敘述。
2. 科學方法排序：提出問題、形成假設、設計實驗、收集與分析、得出結論、表達分享。
3. 完整物理量：數值與單位。
4. MKS 制：m、kg、s。
5. SI 七基本量：長度、質量、時間、電流、熱力學溫度、物量、光強度。
6. 基本量與導出量：力、壓力、功、能量、功率、電荷皆為導出量。
7. 單位代號配對：s、m、kg、A、K、mol、cd。
8. 導出單位：N、J、W、Pa、C。
9. 前綴詞：k、M、G、m、μ、n、p、f。
10. 單位換算：km to m、MHz to Hz、nm to m、Å to m、TB to GB。

## 進階題種子

1. 由定義推導單位：壓力、動能、功率、比熱。
2. 精確數量級：用 `sqrt(10) ~= 3.16` 作界線。
3. 物理史排序：哥白尼、法拉第、馬克士威、普朗克。
4. 人物配對：托勒密地心說、哥白尼日心說、克卜勒橢圓軌道、牛頓三大定律、普朗克量子論。
5. 近代物理分水嶺：1900 年普朗克量子論。
6. 近代物理兩大基石：量子力學與相對論。
7. 廣義相對論例子：預測光線受重力彎曲，1919 日全食觀測支持。
8. 水三相點：舊克耳文定義與新 SI 修訂。

## 混合題或簡答題種子

1. 說明「課本寫了就不用檢驗」違反科學態度的原因。
2. 解釋為何科學理論需要可檢驗的預測。
3. 用 `P=F/A` 推導壓力 SI 基本單位。
4. 將 `250 MHz`、`0.53 Å`、`10 nm` 轉成指定單位。
5. 以表面積／體積比解釋奈米材料性質改變。
6. 說明 SI 單位為何逐漸改用基本物理常數定義。

## 易錯提醒

- g 不是 SI 質量基本單位，kg 才是。
- °C 不是 SI 熱力學溫度基本單位，K 才是。
- C 是庫侖，為電荷導出單位；A 才是電流基本單位。
- `m` 可能代表 meter，也可能代表 milli 前綴；題目語境要清楚。
- `M` 是 mega，`m` 是 milli，大小寫差異很重要。
- `Pa = kg/(m s^2)`，不是 `kg m/s^2`。
- `W = J/s = kg m^2/s^3`。
- 科學理論不是不能改，反而會隨新證據修正。

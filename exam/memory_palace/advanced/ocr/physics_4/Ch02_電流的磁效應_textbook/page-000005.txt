= 磁場與磁力線
法拉第提出磁力線 (line of magnetic force ) 的概念,運用簡單的圖像來描
述空間中磁場的分布。如圖 2-2(1) 所示,將磁棒放在桌上,並在其四周撒上
鐵粉,此時觀察鐵粉的排列即可看出磁力線的分布情形。磁力線由磁棒的 N
極出發,經磁棒外部到磁棒的S極,再從 S 極經磁棒內部回到 N 極。正如可
以用測試電荷偵測空間中某處的電場,我們可以利用小磁針來偵測該處的磁
場 : 小磁針 N 極方向即為磁場方向,小磁針受力量值正比於該處磁場量值。
整理磁力線的特性如下:
1. 磁力線永遠形成封閉曲線:在磁鐵外部由 N 極處出發,回到S 極處,在磁
鐵內部則由 S 極再回到 N 極。
2. 磁力線的切線方向即為磁針 N 極在該處的方向,代表磁場的方向。
3, 磁力線的密度與小磁針在該處所受磁力成正比,代表磁場量值。
4 磁力線不會相交。
依照上述的規則,可以利用磁力線將抽象的磁場視覺化。如圖 2-2(2) 所示 ,
將兩磁棒S 極相互靠近時 ,其磁力線互相排擠,可以看出相同磁極間的斥
力。若將S 極換成 N 極的話 ,其磁力線分布相同,只是方向會反過來。若將
一磁棒的 N 極接近另一磁棒的S 極時,會產生許多相互連接的磁力線,由此
可以看出相異磁極間的吸力。

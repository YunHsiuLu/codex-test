這是高二、高三選修物理 I-V 的跨冊記憶宮殿。出選修物理考卷時，先查本資料夾，再依需要回查 `memory_palace/elementary/global/` 的高一基礎概念。

使用順序：

1. 讀 `concept_map.md`，確認跨冊主軸。
2. 依範圍讀 `chunks/`。
3. 用 `../source_map.md` 追溯原始 PDF 與 OCR。PPM／PNG 渲染中間檔已刪除。
4. 用 `question_design_matrix.md` 設計題型與迷思選項。

目前已納入：

- 選修物理 I：測量與不確定度、直線運動、平面運動、牛頓運動定律、週期運動、萬有引力。
- 選修物理 II：靜力平衡、動量與角動量、功與能量、碰撞、熱學。
- 選修物理 III：波動、聲波、幾何光學、物理光學。
- 選修物理 IV：靜電學、電流的磁效應、電磁感應。
- 選修物理 V：電路學、量子現象、原子結構。

與高一基礎的關係：

- 高一 `foundation` 是概念入口與定性版本。
- 選修 `advanced` 補上更完整的數學模型、向量處理、守恆律、場論、波動光學與近代物理。
- 同名概念不覆蓋高一資料，而是以進階 chunk 延伸。

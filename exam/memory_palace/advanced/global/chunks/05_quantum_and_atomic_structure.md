# 量子現象與原子結構

## 量子現象

- 普朗克能量量子化解釋黑體輻射問題。
- 光電效應顯示光具有粒子性；光子能量 `E = hf`。
- 截止頻率由金屬功函數決定；頻率影響最大動能，強度主要影響光電子數。
- 截止電壓可用來量測光電子最大動能。
- 康普吞效應支持光子具有動量。
- 德布羅意提出物質波，電子繞射與雙狹縫實驗支持粒子也有波動性。

## 原子模型

- 湯姆森發現電子並提出早期原子模型。
- 拉塞福金箔散射指出正電荷與質量集中在原子核。
- 拉塞福模型無法用古典電磁學解釋原子穩定與不連續光譜。
- 波耳氫原子模型引入穩定態與角動量量子化。
- 電子在定態軌道不輻射；躍遷時吸收或放出光子，滿足 `hf = Delta E`。

## 波耳模型的進階重點

- 庫侖力提供電子圓周運動所需向心力。
- 角動量量子化使軌道半徑、速率、週期與能量皆呈離散值。
- 氫原子能階可解釋發射與吸收光譜。
- 波耳模型對氫原子有效，但不是完整的現代量子力學模型。

- 需要選修 IV 的庫侖力、電位能與電磁波概念，才能建立原子模型與光子能量觀。

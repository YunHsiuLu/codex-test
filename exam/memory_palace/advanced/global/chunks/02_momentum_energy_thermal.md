# 守恆律、平衡與熱學

## 靜力平衡

- 質點平衡要求合力為零。
- 剛體平衡除合力為零，也要求合力矩為零。
- 力矩大小與力、力臂相關；方向常用右手定則判斷。
- 支點選擇會影響計算簡便性，但不影響物理結果。

## 動量

- 動量是向量，`p = mv`。
- 牛頓第二定律可寫成合外力等於動量變化率。
- 衝量等於動量變化，力－時間圖面積代表衝量。
- 系統所受外力衝量為零時，系統總動量守恆。

## 角動量

- 角動量是轉動版的動量，與位置向量和動量相關。
- 外力矩合為零時，角動量守恆。
- 天體運動、碰撞偏轉、波耳模型中的角動量量子化都會用到角動量概念。

## 功與能量

- 功能定理：合力作功等於動能改變。
- 保守力可定義位能；非保守力作功造成機械能改變，但總能量仍守恆。
- 位能零點可任意選取，位能差才有直接物理意義。
- 功率描述能量轉換率。

## 碰撞

- 碰撞過程常因內力遠大於外力而近似動量守恆。
- 彈性碰撞：動量與動能皆守恆。
- 非彈性碰撞：動量守恆，但部分動能轉為內能、聲、形變等。
- 完全非彈性碰撞：碰後物體黏在一起，動能損失通常最大。

## 熱學

- 熱是因溫度差造成的能量轉移，內能是系統微觀能量總和。
- 熱力學第一定律連結吸熱、作功與內能變化。
- 理想氣體狀態方程連結壓力、體積、莫耳數與絕對溫度。
- 熱機效率、循環過程與能量流向是常見整合題。

# 選修物理出題設計矩陣

## 適合單選的基礎判斷

- 不確定度表示法與有效數字。
- 速度、加速度、位移圖像判讀。
- 拋體最高點速度與加速度方向。
- 向心力來源與向心加速度方向。
- 力矩方向、平衡條件、角動量守恆條件。
- 電場、電位、電位能正負與方向判斷。
- 磁力方向、感應電流方向、冷次定律。
- 干涉條件、折射頻率不變、駐波節點腹點。
- 光電效應中頻率、強度、截止電壓的影響。

## 適合複選的概念整合

- 同一運動情境同時判斷力、能量、動量是否守恆。
- 斜拋運動中水平與鉛直分量的獨立性。
- 圓周運動與萬有引力對衛星軌道的限制。
- 碰撞題中動量守恆、動能是否守恆、恢復係數。
- 熱學題中內能、作功、吸熱與熱力學第一定律。
- 電磁感應題中磁通量變化、感應電流方向、能量來源。
- 量子題中波動性、粒子性、能量量子化與原子光譜。

## 常見迷思選項

- 不確定度越小，測量值就一定越接近真值。
- 斜拋最高點加速度為零。
- 圓周運動速率不變時，加速度也為零。
- 靜力平衡只要合力為零，不需要合力矩為零。
- 動量守恆代表動能一定守恆。
- 有摩擦或非保守力時，總能量守恆不成立。
- 聲波或光波折射時頻率改變。
- 電場為零處電位一定為零。
- 磁場對靜止電荷有磁力。
- 只要有磁場就一定有感應電流。
- 光強度增加一定提高光電子最大動能。
- 波耳模型是完整正確的現代原子模型。

## 混合題方向

- 實驗數據題：不確定度、圖線斜率、物理量單位與模型假設。
- 運動情境題：斜面、滑輪、摩擦、圓周與能量守恆混合。
- 天體題：萬有引力、圓周運動、能量與週期。
- 剛體題：力矩平衡、質心、支撐力分配。
- 碰撞題：動量守恆、動能變化、彈性或非彈性分類。
- 波動題：駐波、干涉、繞射、聲波共鳴與都卜勒。
- 電磁題：帶電粒子軌跡、導線受力、電磁感應與電路負載。
- 近代物理題：光電效應、物質波、波耳能階與光譜。

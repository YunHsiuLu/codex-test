# 選修物理 Memory Palace 預留區

此區尚未匯入講義。未來用於高二、高三選修物理 I-V，定位為進階物理層級。

規則：

- 不把選修物理內容直接混入高一 `global/`。
- 選修內容可引用高一基礎概念，但要用 `depends_on` 或銜接段落標明。
- 若同一概念在高一與選修都出現，選修區只補進階深度，不改寫高一基礎版本。

建議未來結構：

- `physics_I/`
- `physics_II/`
- `physics_III/`
- `physics_IV/`
- `physics_V/`
- `ocr/physics_I/` 到 `ocr/physics_V/`
- `global/`：選修物理內部的跨冊概念索引。

# 跨章節概念圖

## 主軸一：尺度、物質與基本交互作用

Ch1 的單位、數量級與科學方法，是閱讀全部章節的工具。Ch2 把物質拆成原子、原子核、電子、質子、中子與夸克，並引入四種基本交互作用：重力、電磁力、強核力、弱核力。

這條線往後延伸：

- Ch3：物體運動中的重力、正向力、摩擦力、彈力，宏觀上用牛頓定律描述；微觀來源多與重力或電磁力有關。
- Ch4：電流磁效應、電磁感應與電磁波，是電磁交互作用的統一描述。
- Ch5：核分裂、核融合與質能互換，涉及原子核尺度與強作用的能量釋放。
- Ch6：波耳模型、能階、原子光譜與光電效應，建立在原子結構與能量量子化上。

## 主軸二：力、運動與場

Ch3 用位置、速度、加速度描述運動，用牛頓三定律與常見力解釋運動改變。Ch2 的重力與電磁力提供力的根源；Ch4 把電場、磁場與電磁波連成一套場的觀點。

跨章連線：

- 萬有引力連接 Ch2 的基本交互作用與 Ch3 的克卜勒行星運動。
- 摩擦力、正向力與彈力雖在 Ch3 作為接觸力處理，微觀本質多與電磁力有關。
- 電磁感應中的感應電流方向遵守冷次定律，本質上也受能量守恆約束，連到 Ch5。

## 主軸三：功、能量與守恆

Ch5 是整冊的能量語言。功是力造成能量轉移的量，動能、位能、熱能、電能、光能、化學能、核能都可互相轉換，但總能量守恆。

跨章連線：

- Ch3 的力學問題可用牛頓定律，也可用功能定理或機械能守恆。
- Ch4 的電磁感應不是「無中生有」產生電能，外力或其他能量來源必須提供能量。
- Ch6 的光子能量 `E = hf = hc/lambda`、電子躍遷 `hf = Delta E`，都是能量守恆在微觀尺度的表現。
- 核能中的質量虧損用 `E = mc^2` 連接質量與能量。

## 主軸四：波、光與量子

Ch4 先建立波動語言：週期、頻率、波長、波速、干涉、繞射、折射、多普勒效應，並指出光是電磁波。Ch6 再補上光的粒子性與物質波，形成波粒二象性。

跨章連線：

- 光的干涉與繞射支持波動性。
- 光電效應支持粒子性：是否能逸出電子主要看頻率是否超過底限頻率，不是只看強度。
- 電子繞射與雙狹縫實驗支持物質也有波動性。
- 原子光譜的不連續性連到能階量子化。

## 出題時的整合方向

- 同一現象可要求學生同時判斷「宏觀描述」、「微觀來源」、「能量轉換」。
- 正誤題適合測迷思概念，例如「強度越大一定能產生光電效應」、「有摩擦力能量守恆就不成立」、「原子核大小等於原子大小」。
- 複選題適合串接人物與發現，例如拉塞福、波耳、法拉第、馬克士威、愛因斯坦。

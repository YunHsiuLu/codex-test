# 物質、原子與基本交互作用

## 必記概念

- 原子半徑量級約 `10^-10 m`，常用 `1 A = 10^-10 m`；原子核半徑量級約 `10^-15 m`。所以原子大部分空間是空的，質量主要集中在原子核。
- 電中性原子中，質子數等於電子數。原子序 `Z` 是質子數；質量數 `A` 是質子數加中子數。
- 同位素：質子數相同、中子數不同，化學性質相近，質量不同。
- 質子、中子不是最終基本粒子，它們由夸克組成。
- 可見光波長遠大於原子尺度，因此一般光學顯微鏡不能直接看見原子。

## 原子模型

湯姆森模型：

- 電子被發現後，湯姆森提出帶負電電子嵌在正電物質中的模型，常稱葡萄乾布丁模型。

拉塞福模型：

- 金箔散射實驗顯示，多數 alpha 粒子幾乎直進，少數大角度偏折。
- 推論：正電荷與大部分質量集中在極小原子核，原子內部大部分是空的。
- 拉塞福模型可解釋原子核存在，但用古典電磁學會遇到電子繞核應持續輻射能量而塌陷的困難，這在 Ch6 由波耳模型與量子化處理。

## 四種基本交互作用

- 重力：長程、吸引、支配天體與星系尺度；宏觀重量來自地球重力。
- 電磁力：長程，電荷與電流相關；日常接觸力、摩擦力、正向力、彈力的微觀來源多與電磁作用有關。
- 強核力：短程，作用在原子核尺度，能克服質子間的電斥力，使原子核穩定。
- 弱核力：與某些粒子衰變、核反應過程相關，例如中子衰變。

## 常見迷思

- 不是四種基本交互作用都遵守平方反比。高一範圍中，重力與靜電力常以平方反比描述；強、弱作用是短程作用。
- 電子繞核主要與電磁力相關，不是強核力。
- 金箔散射估出的是原子核尺度，不是原子平均大小。
- 摩擦力不是一種新的基本交互作用，它是宏觀接觸作用，微觀上多與電磁力有關。

## 跨章連結

- Ch3：重力進入自由落體、拋體與行星運動；接觸力進入牛頓定律。
- Ch4：電磁交互作用展開成電場、磁場、電磁感應與電磁波。
- Ch5：強核力與原子核結構連到核能、質量虧損與 `E = mc^2`。
- Ch6：拉塞福模型的限制引出波耳模型、能階與原子光譜。

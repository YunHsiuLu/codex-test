# Memory Palace

這是物理講義的出題記憶宮殿。資料已整理成兩個主要資料夾：

- `elementary/`：高一物理（全），基礎物理範圍。
- `advanced/`：高二、高三選修物理 I-V，進階物理範圍。

## 工作原則

- 高一考卷：先查 `elementary/global/`，再查 `elementary/ocr/`。
- 選修物理考卷：先查 `advanced/global/`，再查 `advanced/ocr/`。
- 混合或銜接題：同時查高一基礎與選修進階，但題目標籤要標明主要層級。
- 若需要精準引用講義原文，回到對應 OCR；若 OCR 不足，再回正式 PDF 來源。
- 渲染圖片中間檔已刪除，不要為一般出題重新渲染 PDF。

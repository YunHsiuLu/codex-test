# Memory Palace

這是物理講義的出題記憶宮殿。現階段已建立的內容屬於「高一物理（全）」範圍，定位為基礎物理；之後若加入高二、高三選修物理 I-V，會放入進階物理層級，不和高一資料混在同一組索引中。

優先入口：

- 層級與標籤規則：`taxonomy.md`
- 高一基礎全域索引：`global/README.md`
- 高一跨章概念圖：`global/concept_map.md`
- 高一來源頁面對照：`global/source_map.md`
- 高一出題設計矩陣：`global/question_design_matrix.md`

高一基礎概念 chunk：

- 物質、原子與基本交互作用：`global/chunks/01_matter_atoms_forces.md`
- 運動、力與場：`global/chunks/02_motion_forces_fields.md`
- 電、磁、電磁波與波動：`global/chunks/03_electric_magnetic_waves.md`
- 功、能量、熱與核能：`global/chunks/04_energy_work_heat_nuclear.md`
- 量子現象、光子與原子模型：`global/chunks/05_quantum_photon_atom.md`
- 跨章節橋接與出題切入點：`global/chunks/06_cross_chapter_bridges.md`

原始資料：

- Ch1 人工整理：`Ch1/`
- Ch2-Ch6 OCR 文字：`ocr/Ch2/` 到 `ocr/Ch6/`
- Ch2-Ch6 章節入口：`Ch2/index.md` 到 `Ch6/index.md`
- 未來選修物理資料預留：`advanced/`

工作原則：

- 高一考卷：先查 `global/`，再查各章來源。
- 選修物理考卷：未來先查 `advanced/`，必要時才回查 `global/` 的高一基礎概念。
- 混合或銜接題：同時查高一基礎與選修進階，但題目標籤要標明主要層級。
- 若需要精準引用講義原文，回到 `ocr/` 對應頁面或 `pdf_png/` 的 PNG 視覺確認。
- 新增章節時，優先補所屬層級的全域 chunk 與跨章橋接，不只建立孤立章節摘要。

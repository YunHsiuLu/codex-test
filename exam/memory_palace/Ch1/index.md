# Ch1 Index：科學的態度與方法

## 章節定位

本章是高中物理入門章，核心不是計算技巧，而是建立：

- 科學態度：理性、不盲從、不妄下決斷、以實驗與觀察檢驗。
- 科學方法：提出問題、形成假設、設計實驗、收集與分析資料、得出結論、表達分享。
- 物理量觀念：完整物理量必須包含數值與單位。
- 單位系統：MKS、SI 七個基本量、導出量。
- 單位換算：SI 前綴詞、科學記號、數量級。
- 物理學發展：古典物理到近代物理，量子論作為約 1900 年分水嶺。

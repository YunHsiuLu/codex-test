# Chunk 02：物理量、SI 與導出單位

來源頁面：page-000003、page-000004、page-000006、page-000014、page-000020

## 物理量

物理學是測量的科學。完整的物理量包含：

- 數值。
- 單位。

例：速度 140 公里／時，`140` 是數值，`公里／時` 是單位。

## MKS 制

力學中最常見三個基本量：

- 長度：公尺，m。
- 質量：公斤，kg。
- 時間：秒，s。

MKS 制即 meter-kilogram-second system。

## SI 七個基本量

| 基本量 | 單位名稱 | 代號 | 相關物理常數 |
|---|---|---|---|
| 時間 | 秒 | s | 銫原子輻射波週期 |
| 長度 | 米／公尺 | m | 光速 |
| 質量 | 千克／公斤 | kg | 普朗克常數 |
| 電流 | 安培 | A | 基本電荷 |
| 熱力學溫度 | 克耳文 | K | 波茲曼常數 |
| 物量 | 莫耳 | mol | 亞佛加厥常數 |
| 光強度 | 燭光 | cd | 發光效能 |

常考：力、能量、功率、壓力、電量不是 SI 基本量，而是導出量。

## 導出量與常見導出單位

導出量：可由基本量依定義或定律導出的物理量。

常見導出單位：

- 電荷：`C = A s`
- 力：`N = kg m / s^2`
- 功或能量：`J = N m = kg m^2 / s^2`
- 功率：`W = J/s = kg m^2 / s^3`
- 壓力：`Pa = N/m^2 = kg/(m s^2)`
- 比熱：`J/(kg K) = m^2/(s^2 K)`

## 推導例子

壓力：

- 定義：`P = F/A`
- `F = ma`，單位為 `kg m/s^2`
- 壓力單位：`(kg m/s^2)/m^2 = kg/(m s^2)`

動能：

- `K = 1/2 mv^2`
- 單位：`kg (m/s)^2 = kg m^2/s^2`

功率：

- `P = W/t = F S/t`
- 單位：`(kg m/s^2) m / s = kg m^2/s^3`

比熱：

- `s = H/(m Delta T)`
- 單位：`J/(kg K) = m^2/(s^2 K)`

巴帕換算：

- 1 百帕 = `10^-3` 公斤重／平方公分。
- 若 1 牛頓 = 0.1 公斤重，則 1 百帕 = `10^2` 牛頓／平方公尺。

## 易錯點

- 質量 SI 基本單位是 kg，不是 g。
- 溫度 SI 基本單位是 K，不是攝氏度。
- 時間 SI 基本單位是 s，不是小時。
- 庫侖 C 是導出單位，不是電流基本單位；電流基本單位是 A。
- 加速度可由長度與時間導出，不是基本量。

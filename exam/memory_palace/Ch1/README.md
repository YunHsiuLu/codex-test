# Memory Palace：高一物理 Ch1 科學的態度與方法

用途：未來出 Ch1 考卷時，優先搜尋本資料夾，不必重新讀取 `pdf_png/Ch1/png` 的講義圖片。

建議搜尋方式：

```bash
rg -n "科學態度|科學方法|假設|實驗" memory_palace/Ch1
rg -n "SI|基本量|導出量|牛頓|焦耳|瓦特|庫侖" memory_palace/Ch1
rg -n "前綴|數量級|奈米|微|毫|公里" memory_palace/Ch1
rg -n "托勒密|哥白尼|伽利略|笛卡兒|牛頓|普朗克|愛因斯坦" memory_palace/Ch1
```

結構：

- `index.md`：章節總覽、頁碼地圖、出題入口。
- `chunks/01_scientific_attitude_method.md`：科學態度與科學方法。
- `chunks/02_units_si_derived.md`：物理量、MKS、SI、基本量與導出單位。
- `chunks/03_prefix_order_magnitude.md`：前綴詞、單位換算、數量級。
- `chunks/04_physics_history.md`：古典物理、近代物理、物理學家與貢獻。
- `chunks/05_extended_contexts.md`：素養題素材、奈米表面效應、SI 修訂與基本常數。
- `question_bank_seeds.md`：出題用關鍵題型種子。

TMUX 中文速查表｜點選本窗格後按 q 關閉
===============================================================================

這是 tmux 的右側指令說明窗格。
左側保留原工作區，右側完整高度顯示本速查表。
點選本窗格後按 q，即可關閉說明。

基本觀念
-------------------------------------------------------------------------------
Prefix（前置鍵）預設是 Ctrl+b。
下方寫成「Prefix c」時，操作方式是：先按 Ctrl+b，放開後再按 c。

工作階段 Session
-------------------------------------------------------------------------------
tmux                         建立新的工作階段
tmux new -s 名稱             建立有名稱的工作階段
tmux ls                      列出工作階段
tmux attach -t 名稱          重新連接工作階段
tmux kill-session -t 名稱    關閉指定工作階段
Prefix d                     暫時離開，但保留工作階段
Prefix s                     顯示並切換工作階段

視窗 Window（類似終端機標籤頁）
-------------------------------------------------------------------------------
Prefix c                     建立新視窗
Prefix n／Prefix p            下一個／上一個視窗
Prefix 0～9                  切換至指定編號的視窗
Prefix w                     顯示視窗清單
Prefix ,                     重新命名目前視窗
Prefix &                     關閉目前視窗

窗格 Pane（同一視窗內分割畫面）
-------------------------------------------------------------------------------
Prefix %                     左右分割
Prefix "                     上下分割
Prefix 方向鍵                切換窗格
Prefix o                     切換至下一個窗格
Prefix z                     放大／還原目前窗格
Prefix x                     關閉目前窗格
Prefix {／Prefix }            向前／向後交換窗格
按住 Prefix＋方向鍵          調整窗格大小

複製與捲動
-------------------------------------------------------------------------------
Prefix [                     進入複製模式，可用方向鍵或 Page Up 捲動
q                            離開複製模式

若已啟用滑鼠：
滑鼠滾輪                    捲動內容並自動進入複製模式
點擊窗格                    選取窗格
拖曳窗格邊界                調整窗格大小
點擊底部狀態列              切換視窗

命令列文字編輯（zsh／shell）
-------------------------------------------------------------------------------
以下快捷鍵直接按即可，不需要先按 tmux Prefix。

Ctrl+A                       將游標移到命令列行首
Ctrl+E                       將游標移到命令列行尾
Ctrl+U                       刪除游標到行首的文字
Ctrl+K                       刪除游標到行尾的文字

常用指令
-------------------------------------------------------------------------------
Prefix :                     開啟 tmux 命令列
:source-file ~/.tmux.conf    重新載入設定
:set -g mouse on             立即啟用滑鼠
:set -g mouse off            關閉滑鼠
:set -g focus-events on      將窗格焦點變化傳給 Codex 等程式
:list-keys                   查看所有快捷鍵
:show-options -g             查看全域設定

通知與 Codex 核准
-------------------------------------------------------------------------------
本工具預設啟用 focus-events，讓 Codex 知道您已切換至其他窗格。
tmux 也會將 BEL 通知傳給外層終端機，而不是只顯示 tmux 訊息。

若 Apple Terminal 仍無通知：
1. 開啟「Terminal → 設定 → 描述檔 → 進階」。
2. 確認「視覺提示」或「聲音提示」等 bell 選項已開啟。
3. 到 macOS「系統設定 → 通知 → 終端機」確認允許通知。

tmux 檔案樹
-------------------------------------------------------------------------------
每個 tmux 視窗最左側會自動建立互動式檔案瀏覽器。
檔案樹會追蹤目前使用中的 split pane，並顯示該 pane 的工作目錄。
資料夾以藍色顯示，預設不展開；最上方的 .. 代表上一層。

Prefix c                     建立新視窗，並自動建立新的檔案樹
Prefix F                     在目前視窗補建檔案樹
cd 路徑                      切換工作目錄後，檔案樹會自動更新
滑鼠單擊／↑／↓               選取項目
雙擊資料夾／Enter            進入資料夾
雙擊 ..／←／Backspace        返回上一層
r／R                         重新追蹤目前 split pane
Space                        預覽檔案內容
滑鼠滾輪                    上下移動清單
Page Up／Page Down           整頁捲動
g／G                         跳到開頭／結尾

手動進入資料夾後，切換其他 pane 不會改變目前路徑；按 r 才恢復自動追蹤。
FILE TREE 上的滾輪會轉換成方向鍵，不會進入 tmux 複製模式。
檔案瀏覽器不會控制其他 pane 中的 Vim，點選一般檔案不會開啟檔案。
Vim 內仍可用 \tt 手動開啟或關閉 NERDTree。

本工具
-------------------------------------------------------------------------------
tmux-help                    在右側完整高度顯示指令說明
tmux-help --popup            在右上角開啟懸浮說明
tmux-help --window           在獨立 tmux 視窗開啟說明
Prefix H                     開啟右側指令說明
Prefix P                     開啟右上角 popup
點選說明窗格後按 q           關閉右側說明

提示
-------------------------------------------------------------------------------
1. tmux 的「視窗」是底部狀態列中的標籤；「窗格」是分割畫面。
2. 暫時不用 tmux 時，建議使用 Prefix d，不必關閉工作階段。
3. 若要把滑鼠選取的文字交給終端機，而不是 tmux，可按住 Option 再拖曳。
4. 檔案樹約占左側 18％，右側說明約占工作區 30％。
5. 說明窗格不會搶走焦點，可繼續操作原工作窗格。
6. 同一個 tmux 視窗只能開啟一個說明窗格。
7. popup 是真正的懸浮視窗，但開啟時無法同時在底下窗格輸入。
